This post is the dimensional source-of-truth for the donor Miata front suspension and scanned pickup coordinates used by the narrowing analysis.

All examples use **205/65R15 tires on 15x6 ET+38 wheels** (the working build spec - see [build constants](./build-constants)), giving a 1230 mm front track and 95 mm of narrowing per side off the donor 1420 mm Miata track. This nearly matches the stock A40 front track of 1232 mm.

> Plots in this post are generated by [`scripts/plot_geometry_reference.py`](https://github.com/HorvathDawson/portfolio-website/blob/main/scripts/plot_geometry_reference.py). All constants and formulas match the analysis below.

## Linkage geometry reference

Every plot and equation in the narrowing posts pulls from the same set of donor dimensions. They are collected here for reference. The donor front subframe has been photogrammetrically scanned (Einstar 2); the scan corrected the UCA pickup geometry and the LCA inner-pivot width that the forum CAD had wrong. Forum-CAD values are kept alongside the scanned values in the Numbers table for diff-ability.

### Glossary

| Term | What it is |
|------|------------|
| LBJ / UBJ | Lower / upper ball joint. Spherical pivot connecting the control arm to the knuckle. The line through them is the kingpin axis. |
| LCA / UCA | Lower / upper control arm. A-arm running from two inner frame bushings to the ball joint. |
| KPI | Kingpin inclination. Inward lean of the kingpin axis in front view. Causes the wheel to rise slightly as it steers (self-centring). |
| ARB | Anti-roll bar. Torsion bar between left and right suspension; resists roll without affecting single-wheel compliance. |
| CL | Vehicle longitudinal centreline. All lateral dimensions are from here. |
| Half-track | Miata: 1420 / 2 = 710 mm. A40: 1230 / 2 = 615 mm. |

Coordinate convention: lateral is measured from car centreline (CL); fore/aft is signed with forward positive; height is signed with up positive. The ball-joint rows use "inboard" (lat from hub centre toward CL), "aft/fwd" (fa from hub centre), and "down/up" (z from hub centre).

### Numbers

Dimensions started as estimates pulled from published Miata data (factory specs, the engineering drawings above, forum measurements, photo-measured CAD). The donor front subframe has now been photogrammetrically scanned (Einstar 2) because the forum CAD produced a static caster of ~11.5 deg - implausible for a street Miata. The **Original** column is the forum-CAD / online-reference value; the **Scanned** column is the corrected measurement from the donor. Where the two agree, only the Original column is filled. The **Source** column tags the Scanned value; the **Original** column was a mix of `published` (factory) and `estimate` (forum / photo CAD).

- `scanned` - donor subframe photogrammetric scan.
- `published` - factory spec or engineering drawing (unchanged).
- `derived` - computed from other rows in this table.
- `build target` - chosen build value, not measured.

The four-bar that drives the front-view kinematics is set by four numbers: the LCA inner pivot, the UCA inner pivot, the LBJ, and the UBJ. The knuckle length and KPI follow directly from LBJ and UBJ. Everything downstream (RC, IC, FVSA, camber gain, scrub, motion ratio) is a function of these four points and the tire radius.
